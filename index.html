Research

I'm interested in robot learning, reinforcement learning, and deploying learned policies on real hardware. My current work focuses on asynchronous RL for vision-language-action models under inference latency.

Learning to Act While Waiting: RL Finetuning of Generalist Robot Policies Under Inference Latency
Brian Zhu, E. H. Harrison, Momen Khalil, Philine Meister, Pranav Atreya, Emanuele Poggi, Philipp Sebastian Schmitt, Eugen Solowjow, Qiyang Li, Georg von Wichert, Finn Ferchau, Gokul Narayanan, Kai M. Wurm, Cesar Colmenero, Yash Shahapurkar, Melih Erdogan, Oier Mees, Andrew Wagenmaker, Sergey Levine, Bernd Kast
Submitted to CoRL, 2026
project page / arXiv

Investigated finetuning action-chunking VLA policies under inference time constraints. Proposed Asynchronous RL with Intermediate Information (ARLI), achieving state-of-the-art performance on simulated and real-world manipulation tasks.

Projects
REAL Real Steel: Human-to-Robot Teleoperation and Punch Planning
E. H. Harrison, Nicholas Q. Bui, Mohamed Homsi, Swetha Rajkumar, Evan Kuo
EECS 106A Final Project, Best Final Project
code / project slides / project website

Engineered a physical harness with IMUs and an ESP32 for teleoperating a UR7e robot via arm movements, supporting pick-and-place and dexterous tasks. Integrated MediaPipe and ArUco detection for hand tracking and built a ROS/RViz GUI for defining and executing robot punches.

Learning to Race: Deep Reinforcement Learning for Autonomous Drone Navigation
E. H. Harrison, Nicholas Q. Bui, Mohamed Homsi, Jared Haertel
EECS 106B Final Project
project slides

Trained a PPO-based drone racing policy with custom reward and termination design in Isaac Sim. Used parallel simulation and curriculum-style resets to improve convergence; achieved the fastest policy improvement rate and fastest average lap time among RL-based teams.

Sah-AI: WandB Experiment-Management Agent With Tool Calling
E. H. Harrison, Swetha Rajkumar, Celine Tan, Katie Wang, Ryan Cheng
EE194-16 Scalable AI: Inference and Serving
code / project paper

Built an agentic AI system that designs and executes ML experiments in WandB from a user's codebase and request. Developed a tool-calling backend using OpenAI's tool API to interact with WandB, read/write code, and summarize results.
